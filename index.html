striped-gradients

159 bytes of striped gradient awesomeness

What is this?

striped-gradients is an experiment to add beautiful visual effects to a website without adversely affecting performance.

npm i -S striped-gradients

This project uses background: linear-gradient(...); to add texture and interest to the page. It goes great with tachyons since it can be combined with different background colors to achieve very interesting results.
